How to get started 

Tourists don’t know where they’ve been, travelers don’t know where they’re going. We help them know their surroundings better and make them an outdoorsy. Wondering how to get started? It's simple, just hit the "Get Started" button in the home page and our Outdoorsy application would display all the outdoor events (Sports & Fitness, Travel & Outdoor) happening around.

How do I enter address to look up events 

Wondering how to search for events? Outdoorsy application allows user to enter address in the "Enter City or Postal Code" textbox. Upon typing partial words of city or postal code, the outdoorsy app displays suggestions making it user friendly with the autocomplete options. Then hit the "Get Started" button to view the outdoor events happening around the specified address location.

I don't know my address, how do I search for events 

Not to worry if you are not sure about your address location. Just hit the "CrossHair" button present in the home page, and the application would identify your location. Then click on the "Get Started" button to view the outdoor events happening around.

Can I sign-up with Facebook account 

Yes ofcourse! Click on the "Sign-Up" button which is available in the top right corner of the home page and then hit the "Connect With Facebook" button from the modal dialog. Authorize the request for the account creation.

Did not receive any activation email 

Check your junk or spam folder to see if the activation email landed there.

How do I edit my profile details 

After successful login, the user should be able to see his/her first and last name on the top right corner of the home page. By clicking on the name, edit profile page appears. The user shall then update his/her profile details.

How do I change my password 

After successful login, the user should be able to see his/her first and last name on the top right corner of the home page. By clicking on the button 'Change Password' on profile page, user's password field appears. The user shall then change his/her password details.

How do I delete my account 

After successful login, the user should be able to see his/her first and last name on the top right corner of the home page. By clicking on the name, edit profile page appears. The user shall then delete his/her profile.

I forgot my password. How do I reset it 

  With the current version of Outdoorsy app, we do not support the reset password feature. It is planned for the next release. Sorry for the inconvenience caused.

What is preferences all about? Why do I need it 

Preferences cater the results displayed over maps. Every user would have their own choices for event categories. Outdoorsy application allows user to filter the results and display events that match their interests. The following preferences are available for user:

  • Search Preference: The search preference should have "Preferred Mode Of Transport" and "Search Radius". The mode of transport is "Driving", "Walking" and "Public Transit". The search radius min value is 2 miles and max value is 50 miles.

  • Travel & Outdoor: The travel and outdoor category has 8 subcategories and are as follows: "Hiking", "Rafting", "kayaking", "Canoeing", "Climbing", "Travel" and "Other"

  • Sports & Fitness: The sports and fitness category has 21 sub categories and are as follows: "Running", "Walking", "Cycling", "Mountain Biking", "Obstacles", "Basketball", "Football", "Baseball", "Soccer", "Golf", "Voleyball", "Tennis", "Swimming & Water Sports", "Hockey", "Motorsports", "Fighting & Martial Arts", "Snow Sports", "Rugby", "Yoga", "Exercise" and "Other".

How do I edit my preferences 

After successful login, click on the blue colored inverted triangle icon on the top right corner of the page and then hit the "Preferences" button. The preferences modal dialog window appears listing all the preferences set by the user. The user has the privilege to edit preferences as per the choice of his/her interest.

How do I locate my location in the maps 

After hitting "GetStarted" button, maps pages is displayed to the user. The user location is displayed in google maps with a blue dot marker having ripple effect around it for easier identification.

What can I view in the maps 

Maps page render the user and events location with the appropriate event marker for classification of the different events. Mouse hovering upon the event marker would display tool tip text which contains ThumbsUp / ThumbsDown icon depending on the user like/dislike preference set for that particular event, name of the event and driving distance in miles from the user location.

How do I view the details of an event 

It's simple! Just click upon any of the event marker that is rendered in the maps page. The outdoorsy application will navigate to the Event Details page displaying the detailed information about the particular event.

How do I share an event in Facebook 

The event details page has a "Facebook Share" button, clicking on which a small popup window opens requesting the user for permission to post the event details in Facebook. After hitting "Share Link" button in the pop up window, the events details are posted on the user Facebook wall.

What does ThumbsUp and ThumbsDown icon and the numbers signify 

The number displayed next to thumbs up icon reflects the number of users who liked the event. And the number displayed next to thumbs down icon reflects the number of users who disliked the event.

Why am I unable to view/post any review comments 

Only registered users of Outdoorsy application gets the privilege to view/post review comments for any events.

How do I purchase tickets for an paid event? What are the Payment methods that are supported 

The event details page has "Ticket Information" section that displays the price of the ticket if it's paid event. Clicking on "Plan My Reservations" button would have the navigation to third party ticket vendor website where the user shall purchase the tickets. Outdoorsy is not liable for any of the monetary transactions happening for the event reservation.

How do I get directions to the event 

The event details page has "Where on Earth" section that displays the direction from user location to the event location. The user has option to select his/her preferred mode of transportation which are Driving, Walking & Public Transit.

Can I get refund for tickets 

Eventbrite organizers set their own refund policies. To see if you can request a refund, first check the event page to see if the event organizer set a refund policy. To request a refund, reach out to the organizer of your event by replying to your order confirmation email. They should respond within 4 business days and once initiated, refunds can take up to 5 business days to show on your account (7 business days outside the U.S.).

How to contact the Organizer 

The contact details of the organizer for an event is available at the bottom of the event details page (The Organizers Section). If you have event-specific questions or want to request a refund, contact the event organizer directly.
